# T013: 企業名の実名表示ポリシー（v1 ドラフト）

> ステータス: **弁護士レビュー前のドラフト**。レビュー（03_legal-risk-memo.md 論点1）を経て確定する。

## 基本原則

**「事実は実名、評価は抑制的に」** — 公表された事実（値上げした・摘発された）は実名で表示し、当社の評価・推測（協調の可能性）には企業名を直接結び付けない。

## 表示レベル定義

| レベル | 情報の性質 | 実名表示 | 表現ルール |
|--------|----------|---------|-----------|
| **L1: 確定事実** | JFTC が排除措置命令・課徴金納付命令を公表した事件 | ✅ 実名 + 事件詳細 | 「JFTC 公表: ○○（命令日・課徴金額）」と出典明記 |
| **L2: 公表事実** | 各社がプレスリリース等で公表した値上げ | ✅ 実名 + 値上げ率・時期 | 事実のみ記載。「協調」等の評価語を個社に付けない |
| **L3: 統計的観察** | 同時値上げの検知結果・リスクスコア | ⚠️ **業界単位で表示** | スコアは「業界」に付与。個社には付与しない |
| **L4: 予測・推定** | 次回値上げ予測・異常検知 | ❌ 業界単位のみ | 「予測であり事実ではない」を明記 |

## 具体的な表現ルール

### ✅ 許容される表現

- 「アイス業界: 2024-04 に 4社が同月に値上げを公表（同時性スコア: 高）」
- 「明治は 2024-04 にアイス各種を +10% 値上げと公表（出典: 同社プレスリリース）」
- 「JFTC は 2023-03、○○ほか3社に排除措置命令（出典: JFTC 公表資料）」

### ❌ 禁止する表現

- 「明治と森永は協調して値上げした可能性がある」（個社への評価の直接付与）
- 「カルテルの疑いがある企業リスト」（L3 以下での断定的リスト化）
- 「X社のカルテルリスク: 87点」（個社へのスコア付与）

## UI 上の必須要素

1. 全ページ共通の免責バナー（既存）を維持
2. L3/L4 の画面には「本表示は統計的観察であり、違法行為の指摘ではありません」を表示
3. 各データ点に出典（L1: JFTC URL / L2: プレスリリース URL / L3-L4: 算出ロジックへのリンク）
4. スコアの算出方法ページ（methodology.html）へ常時リンク

## 異議申立て対応

| 段階 | 対応 | 期限 |
|------|------|------|
| 受付 | 専用フォームで受付、受領通知を自動送信 | 即時 |
| 一次回答 | 根拠データ（出典 URL）を提示 | 2営業日 |
| 事実誤認があった場合 | 該当データを修正 + 修正履歴を公開 | 判明後 1営業日 |
| 事実は正しいが表現に異議 | 本ポリシーに照らして判定。L3/L4 逸脱があれば修正 | 5営業日 |
